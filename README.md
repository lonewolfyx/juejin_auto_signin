<h1 align="center">掘金自动签到助手</h1>

## 使用

### 克隆项目

```shell
git clone git@github.com:lonewolfyx/juejin_auto_signin.git
```

### 安装相关依赖包

```javascript
npm install
```

### 执行签到

```javascript
npm run sign
```
